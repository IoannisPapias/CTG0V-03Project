Οδηγίες Εγκατάστασης του Project
--------------------------------

1. Data Import στη MySQL του Dumpclinicaltrials.sql.
2. Import Project στο Eclipse.
3. Στo ConnectDB.java γίνεται αλλαγή των: connectionURL, username και password με τα αντίστοιχα του χρήστη.
4. Εγκατάσταση και start του apache tomcat.
5. Είσοδος στην εφαρμογή με την αρχική σελίδα home.html, μέσω browser και localhost.


Η εφαρμογή υλοποιήθηκε με χρήση:
--MySQL 8.0.20
--Eclipse IDE 2020-06 (4.16.0)
--Tomcat v8.5 Server
